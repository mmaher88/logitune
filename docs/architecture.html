How to Extend

New Device

1. Create src/core/devices/FooDescriptor.cpp
2. Implement IDevice interface
3. Register in DeviceRegistry constructor
4. Add device images to QRC
Zero changes to existing code

New Desktop Environment

1. Create src/core/desktop/GnomeDesktop.cpp
2. Implement IDesktopIntegration
3. Add detection logic in AppRoot::init()
One line change in AppRoot

New Input Method

1. Create src/core/input/LibeiInjector.cpp
2. Implement IInputInjector
3. Select in AppRoot::init()
One line change in AppRoot
